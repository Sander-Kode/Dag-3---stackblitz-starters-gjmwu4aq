# Guide til webutvikling

Header

- Logo
- Søkemotor/søkefelt (Vi lager visuelt i frontend, ikke funksjonelt)
- Innlogging (Vi lager visuelt i frontend, ikke funksjonelt)
- Navigasjon = tekst eller bilde ut i fra målgruppe
- SoMe = mer tilgjengelig i header enn footer

Main

- Viktigst å få med seg = hero
- Hovedmålet med siden = om deg/bedriften, fremvisning av produkt/prosjekt
- Gjenvisning/Påminnelse om informasjon andre steder på siden i form av f.eks sticky header
- Main er hovedsaklig landingssidene til navigasjon

Footer

- Logo
- Videre navigasjon for mer informasjon
- SoMe = kun i footer kan være for mye "scrolling"
- Kontakt med adresse for besøk
- Kreditering - eier og/eller produsent av siden
- Personvernerskæring (For mer offisielle sider)
